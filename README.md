# hyouka.js

A simple functional programming language interpreter library in node.js

## Install


~~~
$ npm install hyouka.js
~~~

## Usage


### sample program -- calculator

~~~
$ npm run hyouka-calc

calc> 1 + 2
1 + 2
3

calc> and(true false)
and(true false)
false

calc> pow(2 10)
pow(2 10)
1024
~~~
